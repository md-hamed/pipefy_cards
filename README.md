## Pipefy Cards
This is my solution to Pipefy's Recruitment Exercise. It connects to Pipefy's API to download, store and display a list of cards.

#### Ruby version
2.5.1

#### Rails version
5.1.6

#### Installation 
Just the normal installation path, nothing mysteriuos.
```
bundle
rake db:prepare
rails s
```
and tada :tada:
